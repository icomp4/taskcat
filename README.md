# TaskCat

Modern task manager built in Rust + Tauri.

## Features
- Hierarchical process monitoring
- Real-time CPU/Memory stats
- Process control
- Dark theme UI

## Stack
- Rust/Tauri backend
- Vue.js frontend
- Sysinfo core

## Dev
```bash
# Install
bun install

# Dev mode
bun tauri dev

# Build
bun tauri build
```
## Status
Early development. Expect bugs and missing features.

## Roadmap
- Network usage monitoring
